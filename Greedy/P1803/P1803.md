# P1803 线段覆盖问题

## 题目链接
[洛谷 P1803](https://www.luogu.com.cn/problem/P1803)

## 算法
贪心算法

## 思路
1.参加的比赛越早结束越好（可以用反证法）<br>
2.依照次序选择比赛，不断更新最后时间

## 注意事项
1.sort排序语法：sort(name,name+n,cmp)<br>
cmp可以自定义，e.g.<br>
bool cmp(const race& a, const race& b)<br>
{<br>
	return a.end < b.end;<br>
}<br>
2.使用了结构体<br>
3.必须使用不断更新的last变量，开始时一直与上一个比赛的结束时间相比，但是上一次比赛不一定被选中，因此这样是错误的。
